- Google Maps API - v3.0
- You will need your own Google Maps API key for map.js/map2.js files.

Steps to create API Key -
- Go to the Google Maps Platform > Credentials page.
 (https://console.cloud.google.com/project/_/google/maps-apis/credentials?utm_source=developers.google.com&utm_medium=referral)
- Go to the Credentials page
- On the Credentials page, click Create credentials > API key.
- The API key created dialog displays your newly created API key.
- Click Close.
- The new API key is listed on the Credentials page under API keys.
  (Remember to restrict the API key before using it in production.)
